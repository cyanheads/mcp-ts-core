# 0.9.11 — 2026-05-28

## Added

- **`skills/code-simplifier/`** — new external skill for post-session code review and cleanup against `git diff`. Modernizes syntax, consolidates duplication, aligns changes with the existing codebase. Includes framework-specific transformations for tool/resource/prompt definitions, the `ctx` pattern, error factories, and idioms.
- **`mcp-ts-core init --help`** / **`-h`** — explicit help flag for the init subcommand. Unknown flags now error out with usage hint instead of being silently treated as the project name.
- **`templates/AGENTS.md`** and **`templates/CLAUDE.md`** — `tool-defs-analysis`, `code-simplifier`, `git-wrapup`, `release-and-publish`, `migrate-mcp-ts-template`, and `api-linter` added to the skills tables.

## Changed

- **`skills/orchestrations/`** replaces **`skills/multi-server-orchestration/`** — restructured into intent-based workflow routing under `workflows/`: `greenfield-build.md`, `maintenance-release.md`, `field-test-fix.md`, `fix-wrapup-release.md`. Single source for universal rules, orchestrator posture, and sub-agent strategy. Sub-agent fanout is now an optional capability; workflows run linearly when fanout isn't available. Internal audience.
- **`skills/polish-docs-meta/SKILL.md`** v2.2 → v2.3 — MCPB `user_config` placeholder default guidance. Hosts pass the literal `${user_config.X}` string through when an optional field is blank without a `default`, crashing strict validators (`z.email()`, `z.url()`, `.regex()`) at lazy config load. Pair with `z.preprocess` to strip placeholders to `undefined` server-side.
- **`skills/report-issue-framework/SKILL.md`** v1.6 → v1.7 — pre-flight step added for documentation/contract requests. Audit `src/`, `docs/`, and `skills/` (external-audience skills are the public contract) plus public exports before filing duplicates.
- **`skills/git-wrapup/SKILL.md`** v1.0 → v1.1 — minor cleanup; the orchestrator-spawn paragraph is gone (orchestrations skill owns that flow now).
- **`changelog/template.md`**, **`AGENTS.md`**, **`CLAUDE.md`** — `agent-notes` frontmatter field documented.
- **Dependency bumps:**
  - `@biomejs/biome` 2.4.15 → 2.4.16
  - `@cloudflare/workers-types` ^4.20260527.1 → ^4.20260528.1

## Removed

- **`skills/multi-server-orchestration/`** — replaced by `skills/orchestrations/`. Internal audience; no consumer action required.
