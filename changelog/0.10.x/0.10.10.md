# 0.10.10 тАФ 2026-06-30

## Changed

- **`YamlParser.parse`** тАФ js-yaml v5 removed `DEFAULT_SCHEMA` and defaults `load()` to the stricter YAML 1.2 `CORE_SCHEMA`; the parser now passes `YAML11_SCHEMA` to preserve the prior YAML 1.1 load semantics (`yes`/`no`/`on`/`off` тЖТ boolean, `!!timestamp` тЖТ `Date`) that LLM-generated YAML relies on.
- **`scripts/fetch-openapi-spec.ts`** тАФ switched to `import * as yaml from 'js-yaml'`; v5 dropped the default export.
- **`biome.json`** тАФ `$schema` `2.5.0` тЖТ `2.5.1`, tracking the `@biomejs/biome` bump.

## Removed

- **`@types/js-yaml`** тАФ js-yaml v5 ships its own type declarations, so the separate `@types` package is gone.
- **`js-yaml` resolution pin** тАФ dropped the `resolutions`/`overrides` `js-yaml: ^4.2.0` entry; the v5 upgrade resolves the dependency tree without it.
